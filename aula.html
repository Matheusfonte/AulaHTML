O que é HTML?

HTML (HyperText Markup Language) é a linguagem padrão usada para criar e estruturar páginas da web.
Ela define a organização do conteúdo, como textos, imagens, links e outros elementos, e como eles devem ser exibidos em um navegador.
Não é uma linguagem de programação, mas sim uma linguagem de marcação que usa tags para definir a estrutura e o conteúdo de um documento web.
